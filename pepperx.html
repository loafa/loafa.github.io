Pepper for Education Yoshimoto Robotics

Project Description

Pepper for education is meant to make programming Pepper easier for elementary-age children by using a language built off of MIT's Scratch. The previous tools available for Pepper's development were either too simplistic (not allowing enough creativity) or too complicated to be used in a classroom setting. This project was meant to provide an alternative.

I was in charge of designing the application architecture and creating the code base, as well as researching the various capabilities of the Scratch language. The application I developed will be used as the base for the final product, scheduled for release soon.

Languages Used: Javascript, Python
